# Puzzle Heal —— A Vibe Coding Project

一个功能丰富的在线拼图游戏应用，使用 React 和 TypeScript 构建的现代 web 应用。

## 📖 项目概述

这是一个基于 Canvas 的拼图游戏，具有拖拽、旋转、辅助涂画等多种交互功能。游戏内置了 35+ 张精选的中国主题图片，支持自定义图片上传，并能够生成精美的拼图完成卡片。

## ✨ 核心功能

### 🎮 游戏模式
- **拼图模式**: 经典的拼图游戏体验，支持拖拽和旋转拼图块
- **辅助模式**: 通过涂画标记区域来帮助完成拼图
- **完成模式**: 展示完成的拼图和游戏统计数据

### 🧩 拼图特性
- **拖拽操作**: 流畅的拖放交互
- **旋转功能**: 支持 0°、90°、180°、270° 四种旋转角度
- **智能吸附**: 拼图块自动吸附到正确位置
- **锁定机制**: 正确放置的拼图块会被锁定
- **锯齿风格**: 真实的拼图块样式，带有凹凸槽口

### 🎨 图片资源
- **内置图片**: 35+ 张精选中国主题图片
  - 🌸 花卉类
  - 🏞️ 风景类
  - 🐱 动物类
  - 🍜 食品类
  - 🏛️ 建筑类
  - 🎭 卡通类
  - 🎮 游戏娱乐类
- **自定义上传**: 支持上传本地图片作为拼图
- **远程资源**: 可以加载远程拼图图片

### 📊 辅助功能
- **涂画工具**: 标记需要重点关注的区域
- **提示系统**: 显示拼图块的正确位置
- **时间追踪**: 记录游戏时长
- **最佳成绩**: 保存最佳完成记录
- **完成卡片**: 导出包含统计信息的精美卡片

## 🛠️ 技术栈

- **前端框架**: React 19.2.0
- **开发语言**: TypeScript 5.9.3
- **构建工具**: Vite 7.3.1 (SWC 插件)
- **样式方案**: CSS 模块化
- **代码规范**: ESLint (严格模式)
- **渲染引擎**: HTML5 Canvas

## ⚙️ 配置选项

游戏支持多种配置选项：

- **网格尺寸**: 可配置行数和列数
- **画布尺寸**: 可调整游戏区域大小
- **吸附阈值**: 拼图块吸附的敏感度
- **随机旋转**: 是否启用随机旋转
- **显示辅助**: 是否显示辅助线条
- **拼图样式**: 支持锯齿和矩形两种风格

## 🎯 游戏玩法

1. **选择图片**: 从内置图片中选择或上传自定义图片
2. **开始游戏**: 拼图块会随机打乱在画布上
3. **拖拽拼图**: 使用鼠标拖动拼图块到正确位置
4. **旋转调整**: 右键点击或使用快捷键旋转拼图块
5. **使用辅助**: 在辅助模式下标记难以识别的区域
6. **完成拼图**: 所有拼图块正确放置后自动完成
7. **保存记录**: 查看并分享你的完成卡片

## 🔧 开发说明

### 组件架构
- **App.tsx**: 主应用，管理游戏状态和切换不同模式
- **PuzzleEngine**: 核心游戏引擎，处理拼图逻辑
- **GlueLayer**: 涂画层，实现标记功能
- **exportCard**: 卡片导出模块

### 性能优化
- 使用 `useRef` 和 `useMemo` 优化渲染性能
- Canvas 渲染采用高效的重绘策略
- 拼图块使用状态管理减少不必要的计算

### 代码规范
- 启用 TypeScript 严格模式
- 遵循 ESLint React 规范
- 使用 React Hooks 最佳实践
